#SSM集成的基础项目，项目使用Maven管理

#MyBatis3.2.8

#Spring4.1.2.RELEASE

项目使用Spring4.1.2.RELEASE+SpringMVC4.1.2.RELEASE+Mybatis3.2.8

项目集成了Mybatis分页插件和通用Mapper插件

项目使用的hsqldb内存数据库，根据需要可以切换为其他支持的数据库
